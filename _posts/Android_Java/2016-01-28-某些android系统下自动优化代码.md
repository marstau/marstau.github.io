在使用xutils库时,某些android系统下如android6.0会自动将如下代码优化掉:

等待网络请求的时候,通过循环来等待直到返回数据才返回代码。

```
		while (!isOk) {
			if (isOk) {
				isOk = false;
				break;
			}
		}
```

直接不执行以上代码，所以作如下修改以防被优化掉:


```
		while (!isOk) {
			Log.i(TAG, "isOk=" + isOk);
			if (isOk) {
				isOk = false;
				break;
			}
		}
```
