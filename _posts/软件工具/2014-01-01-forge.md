## Commands


```
forge test --match-test test_MinimalSetup -vvvv
forge test -vvv --match-path test/test_MinimalSetup.t.sol
forge build --force
```

```
forge script script/CreatePool.s.sol \         
    --rpc-url $SEPOLIA_RPC_URL \
    --private-key $SEPOLIA_PRIVATE_KEY \
    --broadcast \
    -vvvv
```
