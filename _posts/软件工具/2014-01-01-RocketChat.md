## Installation


#### [Rocket.Chat](https://developer.rocket.chat/open-source-projects/server/server-environment-setup/mac-osx)

compile version: 6.0.1

```
cat package.json | grep -A4 engines | grep node
nvm install x.x.x

cd apps/meteor
meteor --version
cd ../..

yarn
yarn build

```
