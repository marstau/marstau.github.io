为了实现“完全让 AI 帮我制作”，请务必按以下配置操作：
开启 Composer (Cmd + I)：不要在对话框（Chat）里写代码，一定要在 Composer 里写，因为它能直接动你的文件。
切换到 Agent 模式：点击选项，确保它是 Agent 而不是 Write。Agent 会自己运行终端命令（比如 go run 或 npm install）。
Cycle Count 设为 4x：既然是 Pro+，给它最高的自主权。它在改完后端后，会自动去改前端，直到任务闭环。
模型组合策略：
初期规划： o3 Pro MAX Only
疯狂搬砖（最常用）： Sonnet 4.5 cg
处理诡异 Bug： GPT-5.2 Extra High（5.2 的修正能力比 5.1 更强）
开启 Max Mode：这能让 AI 拥有“上帝视角”，看到你所有文件的关联。
总结你的最佳模型清单：
最强思维： o3 Pro MAX Only
最强写手（主力）： Sonnet 4.5 cg
最强全局记忆： Gemini 3 Pro
备用攻坚手： GPT-5.2 Extra High
特别建议： 既然你在开发网站，利用 Pro+ 的优势，多尝试 Sonnet 4.5 cg。这个模型目前的代码生成质量和逻辑连贯性是所有模型中最适合“一键成站”的。
