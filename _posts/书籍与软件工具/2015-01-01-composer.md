## 安装[More](https://docs.phpcomposer.com/01-basic-usage.html)

```
curl -sS https://getcomposer.org/installer | php
mv composer.phar /usr/local/bin/composer

composer selfupdate # 版本更新
composer install # 安装依赖包
```
