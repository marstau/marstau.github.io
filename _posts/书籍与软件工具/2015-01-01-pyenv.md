## COMMANDS

#### 安装

镜像安装
```
brew install pyenv
pyenv install --list
wget http://mirrors.sohu.com/python/3.9.6/python-3.9.6.tar.xz -P ~/.pyenv/cache/;pyenv install 3.9.10
```

#### MAC切换版本
```
pyenv versions
# 当前目录及其目录切换
pyenv local 3.9.10 && export PYENV_ROOT=~/.pyenv && export PATH=$PYENV_ROOT/shims:$PATH
# 验证一下是否切换成功
python -V
```
