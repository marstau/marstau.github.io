## Mac下环境搭建


### 安装

```
brew install mongodb-community
```
### 手动创建db目录

```
mkdir -p /data/db
sudo chown -R marstau /data/db
```

### 启动mongodb服务

```
mongod --dbpath /data/db
```

### 进入mongoDB后台

```
mongo
```

### 卸载

```
brew uninstall mongodb-community
```

## 命令

启动： `brew services start mongodb/brew/mongodb-community`
