## S3使用策略将上传的文件设为公开

In yourbucketname->权限->存储桶策略

```
{
    "Id": "Policy1527490047421",
    "Version": "2012-10-17",
    "Statement": [
        {
            "Sid": "Stmt1527490045591",
            "Action": "s3:*",
            "Effect": "Allow",
            "Resource": "arn:aws:s3:::yourbucketname/*",
            "Principal": "*"
        }
    ]
}
```
