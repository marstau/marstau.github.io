## Installation

```
git clone git@github.com:flutter/flutter.git
export PATH=$PATH:"$HOME/softwares/flutter/bin"

# using mirror
export PUB_HOSTED_URL=https://pub.flutter-io.cn
export FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn

flutter doctor
```
