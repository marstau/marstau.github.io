#### 注释使用

允许如下使用:

```
//每日任务 
message QueryDailyMission 
{
	repeated RewardInfo check = 1; // 连续签到奖励
}
```

不允许如下使用:

```
message QueryDailyMission 
{
	// 注释内容 
}
```
