## 基础语法


#### vw[More](http://www.css88.com/book/css/values/length/vw.htm)

相对于视口的宽度。视口被均分为100单位的vw

```
h1 {
	font-size: 8vw;
}

如果视口的宽度是200mm，那么上述代码中h1元素的字号将为16mm，即(8x200)/100。
```
